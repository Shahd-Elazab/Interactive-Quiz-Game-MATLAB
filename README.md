# Interactive Quiz Game (MATLAB GUI)

This is an interactive quiz game built in MATLAB using GUI elements. It challenges players with engineering-related questions across three difficulty levels: **Easy**, **Medium**, and **Hard**.

---

## Features

- Multiple Difficulty Levels: Easy, Medium, Hard.
- Hints System:
  - Hint 1: Shows a clue.
  - Hint 2: Eliminates two wrong options.
- Countdown Timer: 15 seconds per question.
- Score Tracking and Performance Graph.
- Sound Feedback:
  - `correct.wav` plays for correct answers.
  - `wrong.wav` plays for wrong answers.

---

## How to Run

1. Open MATLAB.
2. Place all files in the same folder:
   - `interactive_quiz_gui.m`
   - `correct.wav`
   - `wrong.wav`
3. Run the game by typing in MATLAB Command Window:
   ```matlab
   interactive_quiz_gui
